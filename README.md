## HTML이란?

인터넷 정보를 웹 브라우저에 보여 줄 때 사용하는 언어

HyperTextMarkupLanguage


html문서의 기본구조 
<!DOCTYPE html>                     // 현재문서가 HTML5언어로 작성된 웹문서라는뜻
<html lang="ko">                    // 웹문서의 시작과 끝을 알리는 태그
<head>                              // 웹문서 해석에 필요한 정보를 입력하는부분
<meta charset="UTF-8">
<title>HTML 기본문서</title>
</head>
<body>                              // 실제로 나타내는 

<h1>프런트엔드 웹 개발</h1>
<hr>
<p>HTML</p>
<p>CSS</p>
<p>자바스크립트</p>
</body>
</html>
